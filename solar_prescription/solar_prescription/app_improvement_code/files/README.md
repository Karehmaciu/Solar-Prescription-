## Data Description
This dataset contains **183 verified Solar Energy Kits** from the VeraSol Product Database.

These products meet either:
- Lighting Global Quality Standards for pico-PV products
- Solar Home System Kit Quality Standards
- IEC TS 62257-9-8 quality standards

## Columns
1. **Brand** - Manufacturer name
2. **Product Name** - Product model name
3. **Model Number** - Specific model identifier
4. **PV Module Maximum Power [W]** - Solar panel wattage
5. **Number of Light Points** - Number of LED lights included
6. **Main Unit Battery Chemistry** - Battery type

## Product Statistics
- **Total Products**: 183 (out of 199 claimed in database - some appear filtered)
- **System Sizes**: 0.3W to 160W
- **Battery Types**: LiFePO4 (majority), Li-Ion, Sealed lead-acid, Lithium-ion
- **Light Points**: 1 to 7 LED lights per system
